# MicromagnetismoTFG-25-26-
Bachelor's Thesis in Physics

This repository contains a micromagnetic simulator based on the Landau-Lifshitz-Gilbert equation with thermal fluctuations.

Implemented features:

Exchange interaction
Magnetostatic interaction
Heun integration scheme
Thermal noise (Brown-Kubo-Hashitsume model)
Hysteresis simulations
Domain formation analysis

The simulator was developed as part of the Bachelor's Thesis:

"Simulation and Analysis of the Magnetodynamics of Magnetic Dipoles using the Landau-Lifshitz-Gilbert Model"

Author: Miguel Ángel Jiménez López
University of Granada
